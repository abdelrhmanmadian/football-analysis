# Football Analysis

Match analysis projects using Python and StatsBomb open data.

## Projects

### UCL Final 2018 — Liverpool vs Tottenham
- Shot map with xG weighting
- Pass map for both teams
- Press map showing intensity zones
- Full tactical analysis

**Key finding:** Liverpool's high press and direct long balls forced Tottenham into 
meaningless short passes in their own half. Salah's second-minute goal changed the 
entire tactical dynamic of the match.

## Tools Used
- Python
- pandas
- mplsoccer
- StatsBomb open data
